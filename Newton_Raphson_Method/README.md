# Newton-Raphson Method for Nonlinear Systems

This folder contains a computational implementation of the Newton-Raphson iterative method designed to find local roots and solve systems of nonlinear equations.

##  Implemented Methodology
* **Iterative Root-Finding:** Executing multi-step approximations ($x_{k+1}, y_{k+1}$) starting from an initial guess to converge toward the exact solution of the system.
* **Convergence Analysis:** Tracking step-by-step state modifications to analyze the quadratic convergence behavior characteristic of Newton's numerical schemes.

##  Technologies Used
* **Python 3**
* **NumPy / Standard Math libraries:** For performing precise scalar and matrix operations during iterative updates.
