# Zero-Sum Matrix Games & Mixed Strategies

This folder contains a computational implementation focused on Game Theory applications, specifically solving non-cooperative zero-sum matrix games using mixed strategy optimization.

##  Implemented Concepts
* **Payoff Matrix Generation:** Constructing randomized matrix formats ($R_1, R_2, \dots, R_7$) representing conflicting gains and losses between competing players.
* **Mixed Strategy Optimization:** Finding the equilibrium vectors where players allocate probability distributions over available actions to maximize their expected payoff.
* **Game Value Estimation:** Computationally evaluating the upper and lower bounds of the game matrix to establish steady game-theoretic balance points.

##  Technologies Used
* **Python 3**
* **NumPy:** For matrix transformations, generating random vector fields, and linear algebraic modeling.
