# Linear Regression Practical Application & Analysis

This folder contains a practical systems modeling task focused on executing and evaluating Linear Regression using matrix transformations and structured experimental data.

##  Implemented Workflow
* **Dataset Structure:** Utilizing multi-feature input matrices ($x_1, x_2, x_3$) along with observed independent trials to model a targeted physical or mathematical system outcome ($y$).
* **Fitted Value Computation:** Generating the estimated responses ($\hat{y}$) to analyze how well the calculated linear plane aligns with the experimental data distribution.
* **Error & Residual Analysis:** Inspecting the variance and deviations between observed values and model predictions to validate the regression assumptions.

##  Technologies Used
* **Python 3**
* **NumPy:** For vector calculations and matrix transformations.
* **Pandas:** For evaluating dataframe variables and structured data logging.
